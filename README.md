# 婚姻家事权益初判 H5

这个文件夹是“微信/即时通信软件可发送链接”的 H5 静态站点版本。

## 文件说明

- `index.html`：H5 主入口，已内嵌样式和脚本，可直接部署。
- `assets/report-preview.png`：页面中使用的 Excel 报告预览图。
- `privacy.html`：隐私政策模板。
- `disclaimer.html`：免责声明模板。
- `部署说明.md`：把 H5 变成在线链接的操作路径。
- `上线前检查清单.md`：正式对外传播前的合规和产品检查项。

## 当前技术状态

- 纯前端静态页面。
- 可部署到任意支持 HTTPS 的静态站点服务。
- 默认只在用户浏览器本地计算和保存，不主动上传用户填写数据。
- 可以通过微信、企业微信、飞书、短信、邮件等发送在线链接。

## 下一步产品化方向

1. 购买或使用现有域名，完成 HTTPS 部署。
2. 补齐运营主体、隐私政策、免责声明、联系方式。
3. 增加“预约律师复核”按钮和后端线索收集接口。
4. 小范围投放，观察打开率、填写完成率、报告生成率、咨询转化率。
5. 需求验证后再迁移为微信小程序或公众号菜单入口。
