# H5 在线链接部署说明

## 目标

把 `H5` 文件夹部署成一个 HTTPS 在线链接，例如：

`https://你的域名/marriage/`

这样用户在微信、企业微信、飞书、短信或邮件中点击链接即可使用。

## 推荐路径

### 路径 A：静态托管，最快上线

适合 MVP 试运行。

1. 准备域名。
2. 选择静态托管服务，例如腾讯云 COS 静态网站、Cloudflare Pages、Vercel、Netlify、GitHub Pages 或自有服务器 Nginx。
3. 上传 `H5` 文件夹内全部文件。
4. 开启 HTTPS。
5. 配置默认首页为 `index.html`。
6. 用微信、企业微信、手机浏览器分别测试打开。

### 路径 B：公众号菜单 + H5

适合内容 IP 和私域转化。

1. 部署 H5 链接。
2. 在公众号菜单、自动回复、文章阅读原文或客服消息中配置链接。
3. 报告生成后引导预约律师复核。

### 路径 C：微信小程序壳 + H5

适合过渡到微信生态。

1. 先部署 H5 链接。
2. 微信小程序使用 `web-view` 承载 H5。
3. 配置业务域名、主体和相关审核材料。
4. 后续逐步把 H5 逻辑迁移为原生小程序页面。

## 部署后必须测试

- 微信内打开是否正常。
- iPhone Safari、安卓微信、电脑浏览器是否正常。
- 页面是否显示完整，不再只显示“选择文件”。
- 风险声明确认是否正常。
- 演示数据、报告生成、复制报告、下载 TXT 是否正常。
- 隐私政策和免责声明链接是否可访问。

## 当前版本的数据边界

当前 H5 默认不上传用户填写内容，只保存在用户本机浏览器。若后续增加预约咨询、材料上传或后台查看线索，需要新增后端接口、数据库、权限控制和隐私授权。
